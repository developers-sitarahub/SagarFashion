## Technical Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS v4
- **Components**: Custom React components
- **Icons**: Lucide React icons
- **Images**: High-quality generated product images
- **Fonts**: Google Fonts (Geist)
